# Django_ASSIGnment!
TO RUN THIS LOCALLY
1.CREATE VIRTUAL ENV
2.CLONE ALL THE FILES
3.OPEN ENV AND TYPE COMMAND:-
run manage.py runserver
4.APPLICATION WILL RUN ON LOCAL HOST PORT 8000.
